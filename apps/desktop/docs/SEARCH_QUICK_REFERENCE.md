# 搜索功能快速参考

## 🔍 快捷键

| 功能 | 快捷键 |
|------|--------|
| 当前文件搜索 | `Ctrl/Cmd + F` |
| 当前文件替换 | `Ctrl/Cmd + H` |
| 全局搜索 | `Ctrl/Cmd + Shift + F` |
| 下一个匹配 | `Enter` |
| 上一个匹配 | `Shift + Enter` |
| 关闭 | `Esc` |

## 📝 当前文件搜索

### 搜索选项
- **🔤 区分大小写**: 精确匹配大小写
- **📖 全字匹配**: 只匹配完整单词
- **🔧 正则表达式**: 使用正则表达式搜索

### 替换操作
- **替换**: 替换当前匹配项
- **全部替换**: 替换所有匹配项

## 🌐 全局搜索

### 搜索范围
- 📄 场景内容
- 👤 角色信息
- 🌍 世界观设定

### 结果导航
- `↑` `↓` 键选择结果
- `Enter` 打开选中项
- 点击结果直接跳转

## 💡 使用技巧

### 1. 快速查找角色
```
全局搜索 → 输入角色名 → Enter
```

### 2. 批量修改名称
```
Ctrl+F → 输入旧名称 → 输入新名称 → 全部替换
```

### 3. 正则表达式示例
- `\d+` - 匹配数字
- `\w+@\w+\.\w+` - 匹配邮箱
- `第\d+章` - 匹配章节号

### 4. 查找情节线索
```
全局搜索 → 输入关键词 → 查看所有相关场景
```

## ⚠️ 注意事项

1. 首次全局搜索会构建索引，需要几秒钟
2. 替换操作可以用 `Ctrl/Cmd + Z` 撤销
3. 正则表达式错误会在控制台显示
4. 全局搜索结果按相关性排序

## 🎯 常用场景

### 场景 1: 修改角色名
1. 打开包含角色的场景
2. `Ctrl/Cmd + H` 打开替换
3. 输入旧名称和新名称
4. 点击"全部替换"

### 场景 2: 查找情节
1. `Ctrl/Cmd + Shift + F` 打开全局搜索
2. 输入情节关键词
3. 浏览所有相关场景
4. 点击跳转到具体位置

### 场景 3: 检查角色出场
1. 全局搜索角色名
2. 查看所有提到该角色的场景
3. 确认角色出场的连贯性

## 🚀 高级技巧

### 使用正则表达式查找对话
```
搜索: ".*?"
说明: 匹配所有引号内的文本
```

### 查找未完成的内容
```
搜索: TODO|待写|待补充
说明: 使用 | 表示"或"
```

### 查找特定格式
```
搜索: 第\d+章
说明: 查找所有章节标题
```
