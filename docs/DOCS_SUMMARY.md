## 🎯 Desktop 项目新功能汇总

### 核心功能
1. **搜索和替换**
   - 当前文件搜索替换（Ctrl+F / Ctrl+H）
   - 全局搜索（Ctrl+Shift+F）
   - 支持正则表达式、区分大小写、全字匹配

2. **侧边栏管理**
   - 可折叠侧边栏（Ctrl+B）
   - 自动保存用户偏好

3. **数据导出**
   - ZIP 结构化导出
   - JSON 完整备份
   - 多种格式支持

### 界面增强
4. **图标主题系统**
   - 6 种预设主题（默认、极简、经典、现代、优雅、作家）
   - 实时预览和切换
   - 自定义文件类型图标

### 内容管理
5. **大纲系统**
   - 章节详情编辑面板
   - 多种视图模式（树形、卡片、图表）
   - 状态管理、剧情点、角色关联

6. **图表系统**
   - Mermaid 图表支持（离线）
   - PlantUML 图表支持（可选）
   - 多种图表类型（结构图、流程图、时间线等）

7. **角色提及功能**
   - @ 符号快速引用角色
   - 自动补全和模糊搜索
   - 悬停显示角色 Wiki
