# 侧边栏快速指南

## 🎯 一句话说明

按 `Ctrl+B` 隐藏/显示书库侧边栏，获得更大的写作空间。

## ⌨️ 快捷键

| 功能 | 快捷键 |
|------|--------|
| 切换侧边栏 | `Ctrl+B` (Mac: `Cmd+B`) |

## 🖱️ 鼠标操作

1. 查看最左侧的 ActivityBar
2. 点击顶部的切换按钮（📖/📕 图标）
3. 侧边栏会滑入或滑出

## 💡 使用技巧

### 技巧 1: 专注写作
```
进入书籍 → 按 Ctrl+B → 隐藏侧边栏 → 开始写作
```

### 技巧 2: 快速切换书籍
```
按 Ctrl+B → 显示书库 → 选择书籍 → 再按 Ctrl+B 隐藏
```

### 技巧 3: 小屏幕优化
```
默认隐藏侧边栏 → 需要时快速显示 → 节省空间
```

## 🎨 视觉提示

- **📖 图标**: 侧边栏当前显示，点击可隐藏
- **📕 图标**: 侧边栏当前隐藏，点击可显示
- **提示文字**: 鼠标悬停查看当前状态和快捷键

## 🔄 状态记忆

你的选择会自动保存：
- 关闭应用后重新打开
- 保持上次的侧边栏状态
- 无需重复设置

## 📐 空间对比

### 隐藏侧边栏
- 写作区域更宽
- 适合专注创作
- 适合小屏幕

### 显示侧边栏
- 快速切换书籍
- 查看书籍列表
- 管理书库

## ⚡ 常见场景

### 场景 1: 开始写作
```
1. 打开一本书
2. Ctrl+B 隐藏侧边栏
3. 专注写作
```

### 场景 2: 切换书籍
```
1. Ctrl+B 显示侧边栏
2. 点击另一本书
3. Ctrl+B 再次隐藏
```

### 场景 3: 浏览书库
```
1. 保持侧边栏显示
2. 浏览所有书籍
3. 管理书库
```

## 🎯 默认行为

- **首次使用**: 侧边栏默认关闭
- **后续使用**: 记住你的选择
- **原因**: 给你更多写作空间

## 🔗 相关功能

- **底部抽屉**: 独立控制，可同时隐藏
- **全屏模式**: 配合使用，完全沉浸
- **命令面板**: 未来可通过命令切换

## 💬 快速问答

**Q: 为什么默认是关闭的？**
A: 为了给你更大的写作空间，特别是在笔记本电脑上。

**Q: 会记住我的选择吗？**
A: 会的，你的选择会自动保存到本地。

**Q: 可以调整侧边栏宽度吗？**
A: 目前还不支持，但已在计划中。

**Q: 有动画效果吗？**
A: 有的，侧边栏会平滑地滑入和滑出。
